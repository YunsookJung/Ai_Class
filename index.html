Chapter 01

📊 매출 데이터 자동합계와 행 강조

학습 목표: 파이썬 코드를 직접 짜지 않고, AI 프롬프트를 이용하여 대용량 엑셀 데이터를 자동으로 요약하고 시각적으로 강조하는 프로그램을 만들어 봅니다.
1단계

준비 작업

  • 작업 폴더 생성:

    내 PC → C드라이브 아래에 새 폴더를 만들고 이름을 ai연습으로 설정합니다.

  • 실습 파일 배치:

    제공된 실습 파일 large_sales_data_1000.xlsx를 복사하여 C:\ai연습 폴더 안에 붙여넣습니다.

2단계

프로그램 및 환경 설치

자동화 스크립트 실행과 빌드를 위해 파이썬 및 AI 도구를 세팅합니다.

파이썬(Python) 설치: 파이썬 공식 웹사이트에서 안정 버전을 다운로드 및 설치합니다. (설치 시 'Add Python to PATH' 옵션 체크 필!)
구글 제미나이(Gemini) 준비: 웹 브라우저로 제미나이에 로그인하여 대화형 프롬프트 작성 창을 띄웁니다.
3단계

AI에게 프롬프트 명령하기

제미나이에 아래 프롬프트 두 개를 차례로 입력하여 최적의 파이썬 스크립트를 도출합니다.

프롬프트 1. 역할 부여 
너는 파이썬 업무자동화 개발전문가야.
프롬프트 2. 요구사항 
C:\ai연습 폴더에는 large_sales_data_1000.xlsx 파일이 있어. 이 파일은 지점별 매출현황 리스트야.
총 매출을 계산해서 맨 아래 행에 추가해주고, 매출이 1000만원(10,000,000) 이상인 행만 노란색 배경으로 강조해주는 파이썬 스크립트를 만들어줘.
저장할 결과 파일 이름은 '지점별_매출현황_리스트-작성일자.xlsx'로 해줘.
4단계

AI 결과 확인 및 문제 해결

AI가 제공한 코드와 설명이 요구사항에 맞는지 검토합니다. 만약 데이터 포맷 오류(예: 매출 금액에 쉼표가 섞여 문자열로 인식되는 등)가 우려되면 추가 질문을 통해 견고한 예외 처리가 들어간 코드를 확보합니다.

5단계

코드 저장하고 실행하기

  1. 라이브러리 설치:

    명령 프롬프트(cmd)를 켜고, 엑셀 조작에 필요한 라이브러리를 설치합니다.

    > pip install pandas openpyxl
  2. 코드 저장:

    AI가 만들어준 파이썬 스크립트를 복사하여 메모장 또는 파이썬 IDE(VS Code 등)에 붙여넣고, 파일명을 매출합계와_행강조.py로 하여 C:\ai연습 폴더에 저장합니다.

  3. 실행:

    명령 프롬프트(cmd)에서 다음 명령어로 스크립트를 구동시킵니다.

    > python C:\ai연습\매출합계와_행강조.py
  4. 결과 확인:

    폴더 내에 지점별_매출현황_리스트-(작성일자).xlsx 파일이 무사히 생성되었는지 확인하고, 가장 하단에 총 합계가 붙었는지, 그리고 1000만원 이상인 행들이 노란색으로 강조되었는지 검증합니다.

Chapter 02

📊 지점별 매출 요약 보고서 만들기

학습 목표: 지점들이 섞여 있는 원본 로우 데이터(Raw Data)를 자동으로 파싱하고 집계하여, 각 지점별 합산 매출 요약 보고서를 완성하는 자동화 스크립트를 구현합니다.
1단계

준비 및 인프라 설치

1장과 동일하게 C:\ai연습 폴더와 실습 파일, 그리고 파이썬 환경이 확보된 상태에서 진행합니다.

2단계

AI 프롬프트 작성하기

대량 데이터 중 '지점명'과 '매출액' 컬럼을 기준으로 그룹 연산(Groupby)을 수행하는 프롬프트를 구성합니다.

프롬프트 1. 역할 부여 
너는 파이썬 업무자동화 개발전문가야.
프롬프트 2. 요구사항 
C:\ai연습\large_sales_data_1000.xlsx 파일에는 여러 지점의 매출 기록이 중복되어서 행별로 섞여 있어.
이 파일 데이터를 읽어서 각 지점명(Branch)별로 총 매출액(Sales)을 합산하여 요약해주는 '지점별 매출요약 보고서'를 만드는 파이썬 스크립트를 작성해줘.
결과 보고서에는 '지점명'과 '합산 매출액' 정보가 담겨야 하고, 저장할 파일 이름은 '지점별_매출요약_보고서.xlsx'로 지정해줘.
4단계

AI 결과 확인 및 문제 해결

AI가 제공한 지점별 매출액 집계(판다스 groupby 연산) 결과에 대해 설명과 작동 방식을 검토합니다.

5단계

코드 저장하고 실행하기

  1. 스크립트 파일 생성:

    AI가 제공한 판다스(Pandas) 기반의 groupby 코드를 복사하여 메모장에 붙여넣고 지점별_매출요약.py로 저장합니다.

  2. 터미널 실행:
    > python C:\ai연습\지점별_매출요약.py
  3. 최종 보고서 확인:

    생성된 지점별_매출요약_보고서.xlsx를 열어 중복 지점들이 병합되고 각 지점별 단일 매출 총액이 정상적으로 연산되었는지 확인합니다.

Chapter 03

📊 지점명별 파일 분리 및 폴더 자동 정돈

학습 목표: 하나의 대용량 엑셀 파일 내 데이터를 지점별로 그룹핑한 뒤, 지점마다 각각 별도의 엑셀 파일로 쪼개어 자동 생성합니다. 더 나아가 날짜 이름의 폴더를 신규 생성하여 파일들을 깔끔하게 정리하는 복합 제어를 구현합니다.
1단계

초기 파일 분리 프롬프트

먼저 지점 열의 유니크한 값들을 기준으로 개별 파일을 뽑아내는 메인 규칙을 요청합니다.

프롬프트 1. 역할 부여 
너는 파이썬 업무자동화 개발전문가야.
프롬프트 2. 요구사항 
large_sales_data_1000.xlsx 파일에서 '지점명' 컬럼을 기준으로 데이터를 분류해서, 각 지점별로 개별 엑셀 파일로 각각 저장해주는 파이썬 코드를 만들어줘.
저장할 파일의 이름 규칙은 '지점명_현재날짜.xlsx'로 해줘.
2단계

코드 수정 및 폴더 생성 기능 확장

💡 업무 노하우: 수많은 지점별 파일이 작업 공간에 쏟아지면 매우 복잡해집니다. 당일 날짜로 된 하위 폴더를 자동으로 생성해 정리하는 구조로 코드를 업그레이드합니다.
피드백 프롬프트. 기능 고도화 
방금 작성해준 코드에서, 분리된 지점별 엑셀 파일들이 C:\ai연습 폴더에 바로 저장되지 않고, 오늘 날짜(예: 20260529) 이름의 새 하위 폴더를 자동으로 만든 다음 그 폴더 내에 일괄 저장되도록 코드를 수정해줘.
4단계

AI 결과 확인 및 문제 해결

AI가 작성한 지점명별 파일 분리 코드에 대해 정상 작동 원리를 살펴봅니다.

5단계

코드 저장하고 실행하기

  1. 스크립트 저장:

    C:\ai연습 폴더에 지점별_파일분리.py 파일로 저장합니다.

  2. 터미널 실행:
    > python C:\ai연습\지점별_파일분리.py
  3. 결과 확인:

    실행 후 작업 폴더에 오늘 날짜(예: 20260530/)로 폴더가 잘 생성되었는지 진입하여 각 지점별(강남점, 강북점 등) 엑셀이 개별 저장되었는지 확인합니다.

6단계

코드 수정 및 폴더 생성 피드백 결과

피드백 프롬프트를 보낸 후 AI가 응답하여 폴더 생성 규칙과 날짜 형식을 고도화해 준 결과를 체크합니다.

Chapter 04

📊 나만의 자동화 GUI 프로그램 & 실행파일(.exe) 제작

학습 목표: 검은색 터미널 화면 대신 마우스 클릭으로 직관적으로 조작할 수 있는 그래픽 사용자 인터페이스(GUI) 윈도우 창 프로그램을 생성하고, 파이썬이 없는 PC에서도 실행 가능한 단독 실행파일(EXE)로 빌드합니다.
1단계

GUI 프레임워크 기반 윈도우 생성

기본적으로 파이썬 내장 라이브러리인 Tkinter를 사용하여 가볍고 강력한 자동화 제어 창을 구현합니다.

프롬프트. GUI 프로그램 기본 설계 
내가 앞에서 완성한 '매출합계 및 행 강조', '지점별 요약 보고서', '지점별 파일 분리' 스크립트 3가지를 마우스로 클릭해서 원클릭으로 작동시킬 수 있는 윈도우 GUI 프로그램을 만들고 싶어.
파이썬 내장 tkinter 라이브러리를 사용하여 예쁜 버튼 3개와 프로그램 로그/진행상태를 실시간으로 보여주는 텍스트 박스창을 포함한 완성형 GUI 스크립트를 제공해줘.
2단계

인터랙티브 기능 추가 (파일 탐색기)

💡 확장 포인트: 하드코딩된 경로 대신, 사용자가 직접 컴퓨터 내 엑셀 파일을 마우스로 찾아와 입력할 수 있는 [파일 찾기] 기능을 탑재시킵니다.
피드백 프롬프트. 파일 로드 UI 추가 
특정 폴더의 파일명 경로가 고정(하드코딩)되어 실행되는 부분을 고치고 싶어.
프로그램 상단에 [파일 불러오기] 버튼을 누르면 파일 탐색기(filedialog) 창이 열려서 분석을 원하는 엑셀 파일을 유연하게 고른 후 각 스크립트 기능이 연동되어 처리되도록 GUI 코드를 수정해줘.
3단계

GUI 툴 로컬 실행 및 GUI 스크립트 보존

AI가 제공한 통합 GUI 완성본 코드를 복사하여 macro_tool.py로 저장한 뒤 정상 작동을 확인합니다.

4단계

단일 단독 실행파일(.exe) 빌드

다른 동료들이 파이썬을 전혀 몰라도 아이콘만 더블클릭하면 툴이 열리도록 빌드 도구를 연동합니다.

프롬프트. 빌드 방법 및 가이드 
방금 작성한 macro_tool.py 파일을 파이썬이 안 깔린 컴퓨터에서도 바로 더블클릭해서 실행할 수 있는 단 하나의 윈도우 실행파일(.exe)로 제작하는 상세 방법과 빌드 명령어를 알려줘.
ℹ️ 핵심 명령어 가이드 (PyInstaller):

명령 프롬프트에서 아래 라이브러리를 먼저 설치하고 변환을 시작합니다.

> pip install pyinstaller
> pyinstaller --onefile --noconsole C:\ai연습\macro_tool.py

빌드가 끝나면 C:\ai연습\dist\ 폴더에 단독으로 동작하는 macro_tool.exe 파일이 탄생합니다!

5단계

공유용 매뉴얼(README) 자동화 제작

동료들이 처음 해당 도구를 전달받았을 때 혼란스럽지 않게 사용 가이드라인을 준비해줄 것을 AI에 요청합니다.

프롬프트. README 매뉴얼 초안 요청 
내가 이 완성된 exe 프로그램을 회사 사내망이나 이메일로 동료들에게 공유할 예정이야.
동료들이 툴을 어떻게 조작해야 하는지 단계별 설명, 올바른 엑셀 양식 예시, 에러가 났을 때 자가 대처 요령 등이 자세히 적힌 텍스트 형태의 'README.txt' 파일 초안을 예쁘게 뽑아줘.
